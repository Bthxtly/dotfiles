This repo contains all my dotfiles managed by [chezmoi](https://github.com/twpayne/chezmoi),
Chezmoi manage dotfile start with `dot_` prefix instead of `.`  for example, the `.zshrc` will be
`dot_zshrc` in chezmoi, so if you don't want use chezmoi, just rename the `dot_` config file to
real `.` file, you can fork and edit before use it.

## How to Usage
1. install `chezmoi` first, if you use `Arch Linux`, simply `sudo pacman -S chezmoi`
2. fork this repo, if you want to manage dotfiles your self, if you just use it, you can just use
   this repo
3. use `chezmoi init https://github.com/bthxtly/dotfiles.git` to init dotfiles
4. use `chezmoi apply -v` to make all dotfiles take effect.
